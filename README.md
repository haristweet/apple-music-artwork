# ジャケット画像ダウンローダー

Apple Music / iTunes の高解像度アートワークを検索してダウンロードできる、単体HTMLの簡易Webアプリです。

🔗 **公開ページ:** https://haristweet.github.io/apple-music-artwork/

## 特長

- **iTunes Search API**（Apple公式・認証不要・無料）を利用
- アルバム / 曲（シングル）をキーワード検索
- 国（日本・米・英・韓・台）と解像度（600〜3000px）を選択可能
- 高解像度（最大3000×3000）のジャケット画像をワンクリックでダウンロード
- インストール不要、`index.html` を開くだけで動作

## 使い方

1. キーワード（アーティスト名・アルバム名・曲名）を入力して検索
2. 一覧から欲しいジャケットを選ぶ
3. 「ダウンロード」で保存、「開く」で原寸表示

## 仕組み

- CORS回避のため iTunes Search API を **JSONP**（`callback`）で呼び出し
- `artworkUrl100`（`.../100x100bb.jpg`）の解像度部分を書き換えて高解像度URLを生成
- 画像本体は `mzstatic.com` から取得しブラウザでダウンロード（失敗時は別タブで開くフォールバック）

## 技術スタック

- HTML / CSS / Vanilla JavaScript（ビルド不要・1ファイル完結）
